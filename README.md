#  DSA Optimization Journey

## 📌 About This Repository

This repository documents my journey of revisiting Data Structures & Algorithms with a focus on **optimization, clean logic, and problem-solving patterns**.

Although I have previously studied DSA academically and achieved an **O (Outstanding) grade**, I realized that most of my earlier solutions relied heavily on **brute-force approaches**.

This repo is my effort to:

* Transition from **brute-force → optimized solutions**
* Strengthen **problem-solving intuition**
* Build **interview-level DSA skills**
